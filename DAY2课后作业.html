一、问答题

  1. JS中常用的数据类型有哪些？

    基本数据类型
      number，string，boolean，null,undefined
    引用数据类型
      object
        普通对象，数组对象，正则对象，日期对象，math等
      function

  2. 浏览器中常用的内核有哪些？

    webkit,gecko,presto,trident

  3. 数据类型之间的区别？

    基本数据类型是指简单的数据段(基本类型值)，引用数据类型是指多个值构成的对象(引用类型值)

  4. 列举常用的关键字和保留字

    常见的关键字：break,do,instanceof,typeof,case,else,new,var,catch,if,let,const
    保留字：int,enum,double,char,float,long等

  5. JS中创建变量的几种常用方式？

    var let const function class import

三、扩展题

  1. 浏览器常用的输出方式，除了console.log还有哪些？

    1.alert('要输出的内容');
    在浏览器中弹出一个对话框，然后把要输出的内容展示出来
    先把要输出的内容首先转换成字符串然后在输出
    2.document.write("要输出的内容");
    直接在页面中展示输出的内容
    3.value 给文本框赋值内容
    document.getElementById('search').value="要给#search这个文本框添加的内容"
    4.innerHTML/innerText 给除了表单元素的标签赋值内容
    document.getElementById('div1').innerText="嘤嘤嘤"
